🪟 Windows 설치가 안 되시나요?

증상을 선택하면 해결 방법을 알려드려요!

Windows Defender SmartScreen 경고
"추가 정보" → "실행" 클릭하면 설치할 수 있어요.
스마트 앱 컨트롤 해제 방법
Windows 보안 → 앱 및 브라우저 컨트롤 → 스마트 앱 컨트롤 → "경고" 또는 "끄기"로 변경하세요.
백신 예외 처리
백신 프로그램에서 오일추 설치 파일을 예외 목록에 추가하거나, 일시적으로 실시간 보호를 끈 후 설치하세요.
반응이 없을 때
.exe 파일을 우클릭 → "관리자 권한으로 실행"을 시도해보세요.

코드 서명 인증서를 준비 중이며, 곧 경고 없이 설치하실 수 있게 됩니다.

🍎 "개발자를 확인할 수 없음" 경고가 뜨나요?

개인 개발 앱이라 아직 Apple 인증서가 없어서 표시되는 경고예요.

1 시스템 설정 → 개인정보 보호 및 보안으로 이동하세요
2 "쌤핀이(가) 차단되었습니다" 옆의 "확인 없이 열기"를 클릭하세요
3 또는: 앱을 Control+클릭 → "열기"를 선택하세요

Apple 코드 서명 인증서를 준비 중이며, 곧 경고 없이 설치하실 수 있게 됩니다.
